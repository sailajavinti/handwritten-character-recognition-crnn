# Handwritten Character Recognition using CNN + CRNN

## 📌 Project Overview
This project is a deep learning-based handwritten digit and character recognition system using CNN models trained on MNIST and EMNIST datasets. It also includes a CRNN architecture for extending the model to word and sentence-level recognition.

---

## 🚀 Features
- MNIST digit recognition (0–9)
- EMNIST character recognition (A–Z)
- CNN-based deep learning model
- CRNN architecture (CNN + BiLSTM + CTC concept)
- Streamlit web application
- Real-time image prediction

---

## 📊 Dataset
- MNIST (Digits)
- EMNIST (Letters)

---

## 🧠 Model Performance
- MNIST Accuracy: 96.47%
- EMNIST Accuracy: 91.74%

---

## 🏗️ Tech Stack

- NumPy
